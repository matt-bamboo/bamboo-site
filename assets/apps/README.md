# App Icon Assets

This folder is reserved for public website app icons.

Expected filenames:

- `runway-icon.[ext]`
- `ignite-icon.[ext]`
- `chalk-icon.[ext]`
- `sparks-icon.[ext]`
- `univoice-icon.[ext]`
- `triptracker-pro-icon.[ext]`
- `match-card-icon.[ext]`
- `allotment-optimizer-icon.[ext]`

Only add standalone icon files that are intended for public website use. Do not add screenshots, design source files, private UI captures, or images containing internal information.
