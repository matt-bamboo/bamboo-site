# Bamboo Holdings Public Site Reference

This folder stores public-safe source packets for Bamboo Holdings app website pages.
The packets are reference material for creating marketing, support, and privacy pages on the public Bamboo Holdings website.

## How to Use These Packets

- Treat every file in this repository as public internet content.
- Use the packets as factual source material when drafting public website pages.
- Keep verified current behavior separate from planned or potential behavior.
- Use `info@bamboo.holdings` as the public contact email unless the owner provides a replacement.
- Avoid claims that an app is publicly available, App Store approved, fully compliant, or privacy-reviewed unless that has been confirmed for the live release.
- Do not add private app code, implementation details, access details, internal strategy, or private user data.

## Privacy and Legal Review

Public privacy pages for sensitive apps need owner and legal review before launch. This is especially important for apps involving financial planning, receipts, children, voice/audio, or cannabis.

Do not claim compliance with COPPA, HIPAA, financial-advice rules, cannabis regulations, App Store privacy labels, or other legal frameworks unless that language has been reviewed and approved.

## Source of Truth

The private app repositories remain the source of truth for product code and current implementation behavior.

The `bamboo-site` repository stores only public-safe marketing, support, privacy reference material, and public website assets. It should not contain private app source code, backend code, auth flows, analytics setup, build tooling, or internal planning documents.
