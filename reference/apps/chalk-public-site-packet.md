## Public Positioning

**App name:** Chalk

**Tagline:** The gymnastics coaching companion

**Short description:**  
Chalk helps gymnastics coaches log sessions, rate skills, and send personalized drill plans to athletes and parents.

**Long description:**  
Chalk brings the daily coaching workflow into one place. Coaches can log individual or class sessions, rate athlete skills on a simple 1–4 scale, and review personalized homework plans generated from those session ratings. Athletes can see homework, track progress, and log practice. Parents get a focused view of homework and session history. Facility admins can manage rosters, classes, programs, curriculum, and coaching content.

Chalk is currently designed as an iOS preview/demo experience with local on-device storage. It does not currently include accounts, cloud sync, or a backend.

## Target Users

Chalk is built for:

- Gymnastics coaches
- Youth gymnasts and athletes
- Parents who want safe visibility into homework and progress
- Gym owners, program directors, and facility admins

## Core Value Proposition

Gymnastics coaching often depends on clipboards, memory, scattered notes, and manually written homework. Chalk gives coaches a structured way to capture what happened in practice and turn it into clear, focused follow-through for athletes and parents.

## Public Feature Summary

### Session logging

Coaches can log training sessions for individual athletes or full classes. Skills are rated on a 1–4 scale and can include coaching notes.

### Class session workflow

Coaches can run class sessions across apparatus or activity areas, rate athletes by station, and complete the class day from one review flow.

### Personalized homework generation

Chalk creates homework drafts from session ratings. Lower-rated skills are prioritized, flexibility is included, duplicate drills are avoided, and plans stay focused.

### Homework review and sending

Coaches review homework before sending. They can remove drills, add a personal message, and set a completion target.

### Athlete progress views

Athletes can view assigned skills, homework, milestones, practice logs, and skill-tree progress.

### Parent visibility

Parents can see homework and session history for their child in a read-only view.

### Curriculum and class management

Facility admins can manage coaches, rosters, programs, classes, custom skills, custom drills, and skill video links.

### Demo mode

Chalk includes a pre-seeded demo gym so users can explore the full workflow without creating an account.

### FAQ

**How do I log a training session?**  
Open the Coach view, choose an athlete or class, rate the relevant skills, add notes if needed, and save the session.

**What do the 1–4 ratings mean?**  
1 means Needs Work, 2 means Developing, 3 means Almost There, and 4 means Nailed It.

**How does homework get created?**  
After a session, Chalk creates a homework draft from the ratings. Coaches can review and adjust the plan before sending it.

**Can coaches edit homework before sending?**  
Yes. Coaches can remove items, add a message, and set a completion target before sending.

**Can athletes check off homework?**  
Yes. Athletes can check off assigned drills, and Chalk stores completion history locally.

**Can parents see homework?**  
Yes. Parents have a read-only view of homework and session history.

**Does Chalk require a login?**  
No. The current preview has no account system. It opens to a role picker.

**Will data sync to another device?**  
No. Current data is stored on the device only and does not sync across devices.

**What happens if I reinstall the app?**  
Locally stored data may be lost if the app is deleted or reinstalled. There is currently no Chalk cloud backup.

**How do I reset the demo data?**  
Use the Facility Admin view, open Settings, and choose Reset demo data.

**Is Chalk available on Android?**  
No. Chalk is currently focused on iOS.

### Known limitations

- iOS only in the current preview
- No account system
- No cloud backup or multi-device sync
- No Android build currently
- No push notifications currently
- No AI processing or machine learning in the homework engine
- Data can be lost if the app is deleted or the device is replaced without backup

## Privacy Page Inputs

Use careful language because Chalk is designed for youth sports and may include information about minors.

### Privacy positioning

Chalk is designed as a local-first coaching tool. In the current preview, data entered into the app is stored on the device and is not transmitted to Chalk servers.

### Data users may enter or create

Chalk may store on the device:

- Athlete names and ages
- Coach names and roster information
- Parent display information in demo or facility data
- Session ratings and notes
- Homework plans and completion records
- Practice logs
- Skill progress and milestone records
- Custom skills and drills
- Facility name, tagline, and location text
- Class, program, roster, and curriculum configuration
- Optional skill video URLs

### Local storage

Current Chalk preview data is stored locally on the user’s device. Chalk does not currently operate a user account system, backend database, or server sync for product data.

### Third-party processing

No third-party AI, analytics, advertising, tracking, or crash-reporting services are used in the current preview.

Optional YouTube links may be entered by coaches or admins as skill video references, but Chalk does not upload user training data to YouTube.

### Accounts

Chalk currently has no account system, login flow, Apple Sign In, Google Sign In, or remote profile.

### AI processing

Chalk does not use AI or external model providers for homework generation. Homework is generated by local rule-based logic.

### Children and minors

Chalk is designed for gymnastics programs and may be used to store information about youth athletes. Coaches, admins, or parents may enter names, ages, skill notes, and training records for minors. In the current preview, this information remains on the device and is not transmitted to Chalk servers.

This section should receive legal review before public launch.

### Payments

Chalk currently has no subscriptions, in-app purchases, or payment processing.

### Analytics and tracking

Chalk currently does not use analytics SDKs, advertising identifiers, behavioral tracking, or third-party crash reporting.

### Data deletion

Users can reset demo data from the Admin settings. App deletion also removes locally stored app data, subject to device backup behavior. There is currently no cloud account to delete.
